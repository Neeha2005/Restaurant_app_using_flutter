# Foodie App

Foodie is a Flutter-based mobile application for food ordering. Users can browse popular dishes, view chef recommendations, add items to their cart, and proceed to checkout seamlessly. The app features a clean UI with gradient banners, responsive design for both mobile and desktop, and smooth navigation between Home, Cart, Checkout, and Order Confirmation screens.

## Features
- Browse popular dishes and chef recommendations
- Add items to cart or buy instantly
- Responsive layout for different screen sizes
- Gradient banners and theme-based UI
- Order confirmation and checkout flow
- Favorites management
